# Question 3 — Reverse Traversal of Linked List

## Data Structures Used
A singly linked list node contains data and a pointer to the next node.

## Functions Implemented
- `push_front()` — Inserts node at beginning.
- `print_list()` — Displays list normally.
- `print_reverse()` — Recursively prints reverse order.

## main() Overview
A linked list is created and displayed in normal and reverse order.

## Sample Output
Normal: 1 2 3 4 5  
Reverse: 5 4 3 2 1
