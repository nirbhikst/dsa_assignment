# Data Structures and Algorithms — Assignment 2

This repository contains implementations for Assignment #2 of the Data Structure and Algorithms course.

## Repository Structure

Each question is implemented in a separate C source file and documented with a corresponding Markdown file:

Programs:
- prog_1.c — Balanced parentheses using stack
- prog_2.c — Infix to postfix conversion and evaluation
- prog_3.c — Reverse traversal of linked list
- prog_4.c — Doubly linked list operations
- prog_5.c — Graph traversal (BFS & DFS)
- prog_6.c — Min heap and max heap construction
- prog_7.c — Dijkstra’s shortest path algorithm
- prog_8.c — Sorting algorithms with statistics

Documentation:
- doc_1.md … doc_8.md — Explanation and sample outputs for each program

## How to Compile and Run

Use GCC to compile any program:

```bash
gcc prog_<q>.c -o prog_<q>
./prog_<q>
